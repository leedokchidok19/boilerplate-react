# boilerplate-react
create react app


# 개발환경
* Node.js : **16.15.0**
* Npm : **8.5.5**
* yarn : **1.22.18**

# 명령어
### 버전 확인
* 노드 버전 확인 : **node -v**
* npm 버전 확인 : **npm -v**
* yarn 버전 확인 : **yarn --version**
### 개발
* **yarn start**

### 설치
* yarn : **sudo npm install -g yarn**
* 리액트 앱 :**yarn create react-app .**
